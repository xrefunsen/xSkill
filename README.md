# xSkill (Paper 1.21.4)

Config tabanlı özel kılıçlar, yetenekler, cooldown ve global seviye/XP sistemi.

## Build

```bash
mvn -DskipTests package
```

Jar: `target/xSkill-1.0.0.jar`

## Commands

- `/xskill give <player> <swordId> [amount]`
- `/xskill reload`
- `/xskill stats [player]`
- `/xskill level [player]`

## Config

`src/main/resources/config.yml`
